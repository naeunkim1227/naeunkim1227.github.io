## 사용자관리 명령어


1. 사용자 추가
    + useradd [옵션] 계정이름
2. 사용자 삭제
    + userdel [옵션] 계정이름
    + 옵션명령어
        - -r 사용자의 홈 디렉터리를 제거
3. 그룹 추가
    + groupadd [옵션] 그룹명
    + 옵션명령어
        - -g 특정 gid 번호로 설정한다.
        - -r 0~1000번대 사이로 gid를 자동으로 설정
4. 그룹삭제
    + groupdel 그룹명
5. 사용자 관리
    + groups 사용자명 : 사용자가 속한 그룹을 보여준다.
6. 사용자 전환
    + su -

## 디렉터리 관리 명령어


1. pwd: 현재 위치한 디렉터리의 절대경로를 출력
2. ls : 디렉터리에 있는 파일의 목록을 표시 *ls -la 자주 사용
    + ls [옵션] [디렉토리] 
    + 옵션명령어
        - l : 파일에 대해서 권한이나 생성 시간처럼 보다 자세한 내용을 출력한다. (long)
        - a : 숨긴 파일이나 디렉터리 등의 현재 디렉터리의 모든 내용을 출력한다. (all)
        - h : 파일 크기를 k, m, g 와 같이 사람이 읽기 편한 단위로 출력한다. (human readable)
        - F : 실행 파일이나 디렉터리 등이 쉽게 구분 될 수 있도록 출력한다.
        - R : 하위 디렉터리의 내용들도 함께 출력한다.
3. mkdir
    + 디렉터리 생성
    + mkdir [옵션] 디렉터리
        - m : 디렉터리의 권한을 지정할 수 있다. 기본값은 755이다.
        - p : 상위 디렉터리가 존재하지 않으면 상위 디렉터리도 만든다
4. rmdir
    + 디렉토리 삭제
    + rmdir [옵션] 디렉터리
        - p : 상위 디렉터리도 지운다. 상위 디렉터리도 비어 있어야 한다.
5. cd
    + 디렉토리 이동명령어

## 파일관리 명령어

**파일의 구분**

- : 일반(보통) 파일

b : 블록 디바이스 파일

c : 문자열 디바이스 파일

d : 디렉토리

l : 심볼릭 링크

p 또는 = : 명명된 파이프(named pipe) / FIFO

s : 소켓(socket)

1. touch
    + 파일의 시간정보 변경
    + touch [옵션] 파일이름
    + 옵션명령어
        - c : 현재 시간으로 파일 시간을 변경한다. (파일이 존재하지 않으면 생성하지 않는다.)
        - d 시간 : 현재 시간 대신 지정한 시간으로 파일 시간을 변경한다. ($ touch ‘2016-02-05 12:00:30’ test)
        - r 파일 : 현재 시간 대신 지정한 파일 시간으로 파일 시간을 변경한다.
        - t MMDDHHMM[[CC]YY][.SS]: 현재 시간대신 지정한 시간으로 변경한다.($ touch -t 201602011000.30 test)
2.  rm
    + 파일삭제 명령어
    + rm [옵션] 파일이름 *rm -rf를 자주사용
    + 옵션명령어
        - r, -R : 일반 파일이 아닌 디렉터리인 경우에는 그 하위 디렉터리와 파일 까지 모두 삭제 한다.
        - i : 파일을 삭제 할 것인지 사용자에게 물어본다.
        - f : -i 옵션을 무시 하고 강제로 삭제한다.
    + 디렉토리 삭제시에는 -rf
    
3. cp
    + 파일 복사 명령어
    + cp [옵션] 원본 사본
    + 옵션 명령어
        - a : 원본 파일의 속성, 링크 정보들을 그대로 유지하면서 복사한다.
        - i : 만약 복사 대상에 같은 이름의 파일이 존재하면 사용자에게 물어 본다.
        - f : 만약 복사 대상에 같은 이름의 파일이 존재하면 강제로 지우고 복사한다.
        - R : 디렉토리를 복사할 경우 그 안에 포함된 모든 하위 디렉터리와 파일들을 모두 복사한다.
        
4.  mv
    + 이름 바꾸기 및 이동
    + mv [옵션] 원본 목적지
    + 파일을 이동하는 명령어, 파일 이름을 바꿀 때도 사용 가능
    + 같은 목적지에서 mv를 사용하면 이름 변경이 가능하다.
    + 옵션 명령어 
        - b : 덮어 쓰게 되는 경우 백업 파일을 만들고 파일을 만든다.
        - i : 덮어 쓸 때 사용자에게 물어 본다.
        - f : 덮어 쓸 때 물어 보지 않는다.

5.  cat
    + 파일의 내용 화면에 출력
    + cat [옵션] 파일이름
    + 보통 옵션 사용하지 않음
    + 옵션 명령어
        - n : 줄 번호를 표시한다.
        - b : 빈 행은 제외하고 줄 번호를 표시한다.
        - E : 각 행마다 끝에 $ 문자를 출력한다.
        
6.  more
    + 화면 단위 분할 후 파일 내용 출력
    + more [옵션] 파일이름
    + 옵션 명령어
        - d : 스페이스나 q키를 누르라는 안내 메시지를 화면에 나타낸다.
        - s : 연속되는 빈 공백 행을 하나의 행으로 출력한다.
    
7. find
    + 특정파일 검색 
    + find [시작 디렉터리] [조건]
        
        find . 현재 디렉토리에서 찾는다
        
        find / 전체에서 찾는다.
        
        fint / -name 'log*' log로 시작하는 모든 문자열 찾기
        
        find / -name '*.log' -print  log 파일 찾기, 후 화면에 출력
        
        find  / -user 'webmaster' 웹마스터가 소유한 파일 모두 출력
        
        find / -perm 명시된 퍼미션으로 된 파일을 찾을 때 사용
        
        find / -type [형태] 형태의 파일을 찾는다.
        
8.  grep
    + 파일 내에서 특정 패턴 검색
    + grep [옵션]  표현 [파일(들)]
    + 옵션 명령어
        - -v 일치되는 내용이 없는 라인 표시
        - -c 일치 되는 내용이 있는 행의 개수 표시
        - -n 일치되는 내용이 있는 행은 행 번호와 함께 표시된다.
    + ps -ef  | grep sshd 를 사용하게 되면 sshd,grep 프로세스가 표시된다
    + ps -ef | grep sshd | grep -v grep 으로 grep 프로스세스 없애기 가능
    
9. 파이프 pipe 
    1. 프로그램의 실행결과를 다른 프로그램의 입력으로 연결한다
    
10. redirection
    + echo "test" > test.txt 파일에 test 추가
    + echo "test2" >> text.txt  
    + 연산자
        - 명령어 > 파일 : 파일이 없으면 생성하고, 있다면 기존의 내용을 지운다.
        - 명령어 >> 파일 : 파일이 없다면 생성하고, 있다면 기존의 내용에 추가한다.
        - 명령어 < 파일 : 파일에서 표준 입력(stdin)을 받는다.
        
        sort < hello.txt > sort.txt
        
        sort < hello.txt
        
        >sort.txt
        
        hello.txt 를 sort 해서 sort.txt 에 저장한다는 뜻
        
    
11.  파일 소유자, 그룹 변경 chown
    + chmod [옵션] 소유자:소유그룹 파일명
    + 옵션 명령어
        - R : 경로와 그 하위 파일들을 모두 변경한다.
        - -help : 도움말을 출력한다.
        - -version : 버전 정보를 보여준다.

## 파일 압축 관리 명령어

1. 묶을때 
    + tar [옵션] 생성파일.tar 묶을 파일
    + 해제 tar [옵션] 파일.tar
    + 옵션 명령어
        - c : 새 저장 파일을 만든다. 즉 묶을 때 사용하는 옵션이다.
        - x : 묶인 파일을 해제 한다.
        - v : 처리 중인 파일을 자세하게 보여준다.
        - f : 파일을 지정한다.
    + 옵션으로 묶을 때는 -cvf, 풀때는 -xvf 자주 사용한다. 
    + gzip 사용시 압축 > tar 가 지원 해줘서, cvfz, xvfz로 사용 한다
